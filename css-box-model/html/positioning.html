The CSS Box Model

The CSS box model is fundamental to layout in CSS.

Each element on the page is represented as a box which has unique properties.

Floats

The float property is used for positioning and layout.One of its most common uses is to wrap text around an element.

The float property has 4 possible values:

  • left - element floated to left of container
  • right - element floated to right of container
  • none - element displayed where it is defined in text
  • inherit - element inherits from parent element

The clear property defines if elements can float beside the cleared element. clear has 5 possible values:

  • left - elements can't be floated on left side
  • right - elements can't be floated on right side
  • none - floats allowed on both sides
  • inherit - element inherits from parent element
  • both - no floating elements allowed alongside

Position Property

The position property defines how an element will be positioned. The property has a top,right,bottom and left property for defining specific positioning.It has 5 possible values which determine how elements position is influennced by other elements.These are:

  • static - element not impacted by side properties
  • relative - element impacted by side properties
  • fixed - When screen is scrolled, element position fixed in window
  • absolute - positioned relative to neareat ancestor
  • sticky - Dependant on user scroll position
