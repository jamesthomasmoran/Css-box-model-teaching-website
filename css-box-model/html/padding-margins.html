The CSS Box Model

The CSS box model is fundamental to layout in CSS.

Each element on the page is represented as a box which has unique properties.

Individual Sides

Padding relates to the space generated within the set border of an element. The padding property can be defined for each individual side of an element. This is done by following the keyword padding with one of the positional values. An example of this is padding-right.

The padding property can be altered using one of three properties:

  • length - defined using units such as px or cm.
  • % - defined using the percentage of the width of the container
  • inherit - padding is inherited from parent element

It is worth noting that padding cannot be set to a negative value.

Margins relate to the space generated around the outside of the set border of an element. THe padding property can be defined for each individual side of an element. This is done by following the keyword margin with one of the positional values. An example of this is margin-left.

The padding property can be altered using the same 3 properties as padding aswell as with the value 'auto'.

The auto value causes the browser to determine the margin. The browser will set the margin so that the element is positioned centrally within its container.

It is worth noting that unlike padding, margins can be set to a negative value.

Shorthand Property

The margin property encompasses all 4 side properties into one instruction. The padding property does the same for the 4 padding sides.

This works by the top property being entered first followed by the right property and then the bottom and left property. An example of this would be: 12px 5px 20px 10px.

This cn be shortened further if 2 oposite sides have the same value. For example, if the left margin equalled 10px and so did the right margin we could enter 12px 10px 20px.

Padding and Element Width

The width of an element specifies the width inside the content area. This is inside of padding, borders and margins. This means that if an elements width is set to 150px, and padding-left is set to 20px, the displayed size will be 170px.

If this is not the desired output, the box-sizing property can be used. Using the value 'border-box', box-sizing sets the displayed width to the value set in the width property of an element
