Border Style

Border style defines how the border appears on each element. The CSS property border-style allows us to define how the border appears.

Each one of the 4 borders of an element can be individually modified using the appropriate property. The top borders style can be specified using the border-top-style property, with the right, bottom and left border requiring border to be followed by the relevant position on the box.

There are a variety of pre-defined Border styles that are allowed in CSS. These are:

  • dashed
  • dotted
  • double
  • groove
  • hidden
  • inset
  • none
  • outset
  • ridge
  • solid

It is important to remember that the default value of border-style is 'none'. This means when styling a border, if the style is not set, no other properties will affect the output of the element.

Border Width

Border Width defines the breadth of each border. The CSS property border-width allows us to define the size of the border.

Each one of the 4 borders of an element can be individually modified using the appropriate property. The top border’s width can be specified using the border-top-width property, with the right, bottom and left border requiring border to be followed by the relevant position on the box.

The border width can be set using a specific size using units such as px or em, but can also use three pre-defined sizes: thin, medium, or thick.

Border Color

Border Color defines the colour of the border. The CSS property border-color sets the color of the border.

Each one of the 4 borders of an element can be individually modified using the appropriate property. The top border’s color can be specified using the border-top-width property, with the right, bottom and left border requiring border to be followed by the relevant position on the box.

The Border colour can be defined using:

  • Name - Specify an accepted color name, for example 'blue'
  • Hex - Specify a hex value, for example #000000
  • RGB – Specify a RGB value, for example rgb(255,255,255)
  • Transparent

If border color is not set then it inherits the color of its element.
